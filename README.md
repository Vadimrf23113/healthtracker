# HealthTracker

Десктопное JavaFX-приложение для ежедневного учета показателей здоровья (вес, вода, сон, заметки) с возможностью
поиска по записям и просмотра базовой статистики. Данные сохраняются локально в JSON-файл.

---

## Описание проекта

HealthTracker предназначен для личного контроля привычек и отслеживания показателей здоровья по датам.

Основные возможности приложения:
- профиль пользователя (имя, возраст, рост, цель)
- ежедневные записи: вес (кг), вода (мл), сон (часы), заметки
- таблица записей с добавлением, редактированием и удалением
- поиск по заметкам и быстрые фильтры
- статистика со средними значениями
- сводка за последние 7 дней
- сохранение и загрузка данных в JSON

Приложение хранит данные локально и не требует подключения к интернету.

## Установка и запуск
### Технические требования

- Java JDK 17
- Maven 3.8+ (или Maven Wrapper из проекта)
- ОС: Windows / macOS / Linux

---

### Запуск через Maven

В корне проекта (где находится `pom.xml`) выполните:

```bash
mvn clean javafx:run
```

## Сборка проекта
Для сборки проекта выполните:
```bash
mvn clean package
```

## Примеры использования

### Добавление или обновление записи

1. Откройте вкладку **Dashboard**
2. Выберите дату
3. Заполните поля:
   - вес
   - вода
   - сон
   - заметка
4. Нажмите кнопку добавления/обновления записи

### Быстрое сохранение

В поле заметок можно нажать `Ctrl + S` для быстрого сохранения данных.

### Поиск и фильтрация записей

На вкладке **Records**:
- введите текст в поле поиска для фильтрации записей
- используйте быстрые фильтры:
  - Есть вес
  - Есть вода
  - Есть сон
  - Есть заметка
